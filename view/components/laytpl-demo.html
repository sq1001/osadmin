模板引擎组件 (laytpl)

laytpl 是 Layui 的模板引擎，支持变量输出、循环、条件判断等功能

基础语法
{{= d.field }} 输出普通文本（不转义）
{{ d.field }} 输出转义文本（防止 XSS）
{{# js语句 }} 执行 JavaScript 语句
转义 vs 非转义
转义输出会将 HTML 标签转为实体，防止 XSS 攻击
变量输出
循环渲染
条件判断
自定义分隔符
使用 laytpl.config() 配置自定义分隔符，避免与其他模板引擎冲突
完整示例：用户列表

使用示例

// 基础用法
var tpl = document.getElementById('template-id').innerHTML;
laytpl(tpl).render(data, function(html){
  document.getElementById('view-id').innerHTML = html;
});

// 同步渲染（返回渲染结果）
var html = laytpl(tpl).render(data);

// 转义输出 {{ }} vs 非转义输出 {{= }}
// {{ d.content }}   - 转义输出，防止 XSS
// {{= d.content }}  - 原样输出

// 循环渲染
{{# layui.each(d.list, function(index, item){ }}
  
  • {{= item.name }}
    • 
{{# }); }}

// 条件判断
{{# if(d.type === 1){ }}
  
    类型1
    
{{# } else { }}
  
    其他类型
    
{{# } }}

// 自定义分隔符（全局配置）
laytpl.config({
  open: '[[',
  close: ']]'
});
// 使用后记得恢复默认
laytpl.config({
  open: '{{',
  close: '}}'
});
